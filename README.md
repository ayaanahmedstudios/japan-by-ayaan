# Japan by Ayaan

https://ayaanahmedstudios.github.io/japan-by-ayaan/

An interactive photographic archive mapping stories and visuals from a trip across Japan. This project was developed for the Graphic Design 2 "Resource Web" assignment (Feb-Mar 2025).

## About
This project serves as a digital archive of photographs taken across Japan. It features an interactive map (SVG) that links to specific stories and metadata regarding locations such as Tokyo, Kyoto, and Osaka.

### Features
* **Standardized Structure:** Clean HTML5 and CSS3 using semantic naming.
* **Optimized Media:** High-fidelity images hosted via Cloudinary for fast loading.
* **Typography:** Custom integration of Departure Mono, Tiny5, and Atkinson Hyperlegible fonts.
